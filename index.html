Projects

Small Radio Telescope (21 cm Hydrogen Line)

A small radio telescope built to observe the 21 cm hydrogen line in the Milky Way. The design is a partial reimplementation of the BHARAT horn antenna research paper; we used a pyramidal horn antenna to observe the hydrogen line. Signal chain: Nooelec H1 Sawbird barebones SDR for analog-to-digital conversion, with RTL-SDR for frequency observation and data capture. Software: SDR# (SDR Sharp).

Radio Astronomy RTL-SDR Nooelec Sawbird Pyramidal Horn SDR#
Geometry Destroyers

Geometry Destroyers

A C++ game built with the SFML library using an Entity-Component System (ECS) architecture. You destroy spawning geometric shapes to score points; when a shape is destroyed it splits into smaller shapes that also award points. One hit from any shape ends the run and resets the game, so avoiding contact while clearing the screen is the challenge.
